# BeeEye - მონაცემთა ინტელექტუალური ანალიტიკის პლატფორმა

**BeeEye** არის ინოვაციური ბიზნეს-ანალიტიკის ინსტრუმენტი, რომელიც შევქმენით GTU ტექნოლოგიური ჰაკათონისთვის 2026. ჩვენი მიზანია გავამარტივოთ მონაცემთა ანალიზი ნებისმიერი ტიპის ბიზნესისთვის და მივცეთ მენეჯერებს საშუალება, ბუნებრივ (ქართულ) ენაზე დაუსვან კითხვები მონაცემთა ბაზას. 

პროექტი აერთიანებს თანამედროვე ხელოვნურ ინტელექტს (LLM) და მკაცრად დაცულ SQL გარემოს, რათა მომხმარებლის კითხვები მომენტალურად გარდაიქმნას სანდო ფინანსურ და ოპერაციულ რეპორტებად.

## ჩვენი ხედვა
ბიზნესში გადაწყვეტილებების მისაღებად მონაცემები სასიცოცხლოდ მნიშვნელოვანია, თუმცა SQL-ის ან რთული BI პროგრამების (PowerBI, Tableau) სწავლა დროს მოითხოვს. BeeEye-ს დახმარებით, მომხმარებელი უბრალოდ წერს: "რა არის ჩვენი შემოსავალი ამ თვეში?" და წამებში იღებს ზუსტ პასუხს, ვიზუალურ გრაფიკს და სტრუქტურირებულ ცხრილს.

## ძირითადი ფუნქციონალი
- **ქართული ენის სრული მხარდაჭერა:** AI აგენტი სრულყოფილად ამუშავებს ქართულ კითხვებს და აბრუნებს გამართულ ქართულ პასუხებს.
- **ავტომატური SQL გენერაცია და ვალიდაცია:** იყენებს Google Gemini-ს ბუნებრივი ენის SQL-ში სათარგმნად. ჩაშენებული `sqlglot` ვალიდატორი ბლოკავს საშიშ ოპერაციებს (DROP, DELETE, UPDATE), რათა უზრუნველყოს ბაზის 100%-იანი უსაფრთხოება.
- **ინტერაქტიული დეშბორდი და ჩათი:** თანამედროვე, სწრაფი ინტერფეისი React-ზე, რომელიც ინახავს საუბრის კონტექსტს და აჩვენებს გრაფიკებს პირდაპირ ეკრანზე.
- **პირდაპირი წვდომის რეჟიმი:** ჩათში `/sql` პრეფიქსის გამოყენებით შეგიძლიათ უშუალოდ დაუკავშირდეთ ბაზას და მიიღოთ მონაცემები ხელოვნური ინტელექტის შუალედური რგოლის გარეშე (იდეალურია დეველოპერებისთვის).

## ტექნოლოგიური სტეკი
ჩვენ ავირჩიეთ თანამედროვე და მასშტაბური ტექნოლოგიები:
- **Backend:** Python, FastAPI, SQLAlchemy, LangChain/Google GenAI
- **Frontend:** React, Vite, Recharts, Lucide Icons
- **Database:** PostgreSQL (ავტომატური სატესტო მონაცემებით - Seed)
- **Infrastructure:** Docker, Docker Compose

## ლოკალურად გაშვების ინსტრუქცია

პროექტის გასაშვებად დაგჭირდებათ Docker და Node.js.

### 1. მონაცემთა ბაზისა და სერვერის გაშვება (Backend)
გახსენით ტერმინალი პროექტის მთავარ საქაღალდეში:
```bash
# 1. დააკოპირეთ გარემოს ცვლადების ფაილი
cp .env.example .env

# 2. გაუშვით კონტეინერები (დაელოდეთ 15-20 წამი ბაზის ინიციალიზაციას)
docker-compose up -d db

# 3. დააინსტალირეთ Python ბიბლიოთეკები
pip install -r requirements.txt

# 4. გაუშვით სერვერი ლოკალურად
uvicorn app.main:app --reload --port 8000
```

### 2. კლიენტის აპლიკაციის გაშვება (Frontend)
გახსენით ახალი ტერმინალის ფანჯარა იმავე საქაღალდეში:
```bash
npm install
npm run dev
```

ამ ნაბიჯების შემდეგ აპლიკაცია ხელმისაწვდომი იქნება მისამართზე: `http://localhost:5173`.
